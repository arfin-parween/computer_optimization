Question 1 : Maximize z = 3x + 2y + 5z
             subject to, x + 2y + z <= 430 
		                3x + 2z <= 460 
					     x + 4y <= 420 
				x, y, z >= 0
Answer :     x = 0, y = 100, z = 230

Input : A = {1, 2, 1},
            {3, 0, 2},
            {1, 4, 0}
                
        b = {430, 460, 420}
        c = {3, 2, 5}
		boolean isMaximization = true;
		
Output : [0.0, 100.0, 230.0]
